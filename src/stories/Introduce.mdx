# Introduce devent-design-system-v2

데벤트 디자인 시스템은 여러 프론트엔드 프로젝트마다 비슷한 컴포넌트를 반복 구현하는게 귀찮아서 개발한 프로젝트 입니다. 이 디자인 시스템은 다음과 같은 목적으로 개발되었습니다.

- 여러 프론트엔드 프로젝트에 들어갈 디자인을 하나로 통합해 시간 절약
- 디자인에 통일성을 부여해서 심미성 강화

또한 다음과 같은 항목에 집중했습니다. 이는 기존 v1과 이번 디자인 시스템이 본질적으로 다름을 의미합니다.

- 내 개인 프로젝트 뿐 아니라 해커톤, 팀 프로젝트, 다른 사람들까지 사용 가능한 범용성
- 쉽게 설치하고 쉽게 적용하면서 소프트웨어 의존성이 적고 가볍게
- 인터렉션과 애니메이션 부분 디테일 추가
- 모바일 사용성 강화
- 확장성 염두
- 타입 스크립트와 유니언 타입의 지정으로 빠른 탐색 가능
- npm 패키지 설치만 하면 별도의 설정 없이 바로 사용 가능하도록
