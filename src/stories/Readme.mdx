# devent-design-system-v2

React/TypeScript 환경에서 사용 가능한 디자인 시스템 입니다. 버전 1과 마찬가지로 효율성과 통일성에 중심을 두었습니다.
버전 1의 시행착오를 딛고 보다 광범위하게 사용할 수 있는 디자인 시스템으로 개발하고자 합니다.

버전 2에서는 여러 프로젝트를 진행하면서 저만의 스타일로 디자인을 풀어내고 있습니다. 모던하면서도 다른 디자인 시스템과는 차별이 된 상태에서 무엇보다 UX와 인터렉티브 애니메이션 측면에도 개선이 있습니다. 이를 통해 버전 1에서 있었던 모바일 사용성 문제와 개성 부족 문제를 한 번에 해결할 수 있습니다.

물론 버전 1에서는 부트스트랩5와 기타 라이브러리를 의존해 사용했습니다. 제가 몇 년 전에 사용한 스택인데 React로 넘어오면서 더 이상 사용하지 않고 있습니다. 한 몇 년 동안에는 디자인 시스템 없이 잘 개발해 왔으나 대학 진학 문제로 개발 시간이 부족해진 점과 이전과는 다르게 더 이상 디자인에 시간을 쏟고 싶은 마음이 없어 개발하게 되었습니다. 무엇보다 AI에게 이 디자인 시스템을 학습시킨 다음 컴포넌트를 뽑아내라고 하는게 더 효율적일 듯 합니다.
